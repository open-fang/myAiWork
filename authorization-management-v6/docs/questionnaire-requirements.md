# 问卷题目配置功能需求文档

## 1. 背景与概述

### 1.1 背景
授权书管理系统中的场景配置需要支持问卷功能，用于定义授权规则中需要填写的问卷内容。当前Scene实体已有`questionnaire`字段，但缺乏完整的问卷题目配置管理功能。

### 1.2 业务目标
- 提供问卷题目的完整生命周期管理（创建、编辑、删除、查询）
- 支持多语言问卷内容维护（中文、英文）
- 支持问卷答案的灵活配置

### 1.3 范围
- 题目管理：题目编号自动生成、题目内容多语言维护
- 答案管理：答案编号自动生成、答案内容多语言维护
- 与场景关联：场景可关联多个问卷题目

## 3. 非功能性需求

### 3.1 性能要求
- 列表查询响应时间 < 2秒
- 弹窗打开时间 < 1秒
- 支持分页，每页默认10条，最大100条

### 3.2 数据约束
- 题目编号格式：QT + 时间戳（如：QT20260326001）
- 答案编号格式：ANS + 时间戳（如：ANS20260326001）
- 题目内容最大长度：500字符
- 答案内容最大长度：200字符
- 每个题目最多配置10个可选答案

### 3.3 安全要求
- 所有操作需记录操作日志
- 删除操作需二次确认

### 3.4 兼容性要求
- 支持主流浏览器（Chrome、Firefox、Edge）
- 响应式设计，支持1920x1080及以上分辨率
