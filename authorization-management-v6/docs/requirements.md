# 授权书管理系统 - 需求分析文档

## 1. 项目概述

### 1.1 项目背景
构建一个授权书管理系统，用于管理授权书的基本信息、场景配置、规则配置等。

### 1.2 系统用户
- **授权书管理员**：负责录入授权书基本信息、维护场景和规则、发布授权书

### 1.3 技术栈
- 前端：Vue 2
- 后端：Java JDK8 + Spring Boot + PostgreSQL

## 2. 用户故事清单

### 2.1 授权书列表页（/AuthLetterList）

#### US-001 查询授权书
**作为**授权书管理员
**我想要**通过多种条件查询授权书列表
**以便于**快速找到目标授权书

**验收标准**：
- 支持按授权书名称模糊查询
- 支持按授权对象层级多选查询
- 支持按适用区域树形多选查询
- 支持按授权发布层级多选查询
- 支持按授权发布组织树形多选查询
- 支持按发布年份查询
- 支持按状态查询
- 支持重置查询条件
- 查询结果分页展示

#### US-002 新建授权书
**作为**授权书管理员
**我想要**创建新的授权书
**以便于**开始录入授权书信息

**验收标准**：
- 点击"新建授权书"按钮跳转到授权书详情页
- 新建的授权书状态默认为"草稿"

#### US-003 批量操作授权书
**作为**授权书管理员
**我想要**批量操作授权书
**以便于**提高工作效率

**验收标准**：
- 支持批量更新、生效、失效、删除
- 操作前需要确认
- 只能操作草稿状态的授权书（发布、生效除外）

#### US-004 查看授权书详情
**作为**授权书管理员
**我想要**点击授权书名称查看详情
**以便于**了解授权书完整信息

**验收标准**：
- 点击授权书名称跳转到详情页
- 根据状态控制页面可编辑性

### 2.2 授权书详情页（/AuthLetterDetail）

#### US-005 维护授权书基本信息
**作为**授权书管理员
**我想要**录入授权书的基本信息
**以便于**建立授权书档案

**验收标准**：
- 必填字段：授权书名称、授权发布层级、授权发布组织、授权对象层级、适用区域、发布年份、内容摘要
- 草稿状态下可编辑，已发布状态不可编辑

#### US-006 管理附件
**作为**授权书管理员
**我想要**上传和管理授权书附件
**以便于**保存授权书相关文档

**验收标准**：
- 支持上传附件（记录docId）
- 支持下载附件
- 支持删除附件
- 支持批量删除
- 表格展示附件列表，支持分页

#### US-007 配置授权规则
**作为**授权书管理员
**我想要**配置授权规则场景
**以便于**实现场景匹配

**验收标准**：
- 支持添加场景
- 支持编辑场景
- 支持删除场景
- 支持批量删除

#### US-008 配置场景规则
**作为**授权书管理员
**我想要**为场景配置详细规则
**以便于**实现精准的场景匹配

**验收标准**：
- 场景名称、产业、业务场景、决策层级、具体规则为必填
- 支持规则条件配置（新增条件、新增子条件组）
- 支持条件之间的且/或切换
- 支持多种对比类型（字段、数值、文本、比例）
- 支持问卷设计
- 规则和问卷至少配置一项

#### US-009 发布授权书
**作为**授权书管理员
**我想要**发布授权书
**以便于**使授权书生效

**验收标准**：
- 草稿状态可发布
- 发布后状态变为"已发布"
- 记录操作日志

#### US-010 失效授权书
**作为**授权书管理员
**我想要**使授权书失效
**以便于**停用已发布的授权书

**验收标准**：
- 已发布状态可失效
- 失效后状态变为"已失效"
- 记录操作日志

#### US-011 查看操作日志
**作为**授权书管理员
**我想要**查看授权书的操作历史
**以便于**追溯变更记录

**验收标准**：
- 已发布状态显示日志区块
- 日志区块默认折叠
- 按时间倒序排列
- 支持分页

### 2.3 规则参数配置页（/RuleParamConfig）

#### US-012 管理规则参数
**作为**授权书管理员
**我想要**配置规则参数
**以便于**在场景规则中使用

**验收标准**：
- 支持按名称、英文名、状态查询
- 支持新建、编辑规则参数
- 支持批量生效、失效
- 表格展示参数列表，支持分页

#### US-013 配置规则参数属性
**作为**授权书管理员
**我想要**配置规则参数的详细属性
**以便于**定义字段的数据来源和类型

**验收标准**：
- 必填字段：名称、英文名、状态、数据类型
- 业务对象和取值逻辑可配置多个
- 数据类型支持：文本、数值、比对字段、比率

## 3. 字段定义清单

### 3.1 授权书实体

| 字段名 | 显示名称 | 类型 | 长度 | 必填 | 说明 |
|--------|----------|------|------|------|------|
| name | 授权书名称 | String | 200 | 是 | 单行文本输入 |
| authObjectLevel | 授权对象层级 | Array | - | 是 | 多选，平铺列表 |
| applicableRegion | 适用区域 | Array | - | 是 | 多选，三层树形 |
| authPublishLevel | 授权发布层级 | Array | - | 是 | 多选，平铺列表 |
| authPublishOrg | 授权发布组织 | Array | - | 是 | 多选，三层树形 |
| publishYear | 发布年份 | Integer | - | 是 | 年份选择器 |
| summary | 内容摘要 | String | 4000 | 是 | 多行文本 |
| status | 状态 | String | 20 | 是 | DRAFT/PUBLISHED/INVALID |

### 3.2 场景实体

| 字段名 | 显示名称 | 类型 | 长度 | 必填 | 说明 |
|--------|----------|------|------|------|------|
| sceneName | 场景名称 | String | 200 | 是 | 单行文本 |
| industry | 产业 | Array | - | 是 | 多选，树形 |
| businessScene | 业务场景 | String | 200 | 是 | 单选，平铺列表 |
| decisionLevel | 决策层级 | String | 200 | 是 | 单选，平铺列表 |
| specificRule | 具体规则 | String | 1000 | 是 | 多行文本 |
| rules | 规则配置 | Object | - | 否 | JSON格式 |
| questionnaire | 问卷设计 | Array | - | 否 | JSON数组 |

### 3.3 附件实体

| 字段名 | 显示名称 | 类型 | 长度 | 必填 | 说明 |
|--------|----------|------|------|------|------|
| docId | 文档ID | String | 100 | 是 | 上传后生成 |
| docName | 文档名称 | String | 500 | 是 | - |
| docType | 文档类型 | String | 50 | 是 | 下拉选择 |
| isEncrypted | 是否加密 | Boolean | - | 否 | 暂不实现 |

### 3.4 规则参数实体

| 字段名 | 显示名称 | 类型 | 长度 | 必填 | 说明 |
|--------|----------|------|------|------|------|
| name | 名称 | String | 200 | 是 | 中文显示名 |
| nameEn | 名称英文 | String | 200 | 是 | 英文编码 |
| businessObject | 业务对象 | String | 200 | 否 | - |
| valueLogic | 取值逻辑 | String | 500 | 否 | JSONPath表达式 |
| status | 状态 | String | 20 | 是 | ACTIVE/INACTIVE |
| dataType | 数据类型 | String | 50 | 是 | TEXT/NUMBER/FIELD/RATIO |

## 4. 下拉列表数据源清单

| 下拉组件 | 类型编码 | 数据类型 | 说明 |
|----------|----------|----------|------|
| 授权对象层级 | AUTH_OBJECT_LEVEL | 平铺列表，多选 | 调用接口获取 |
| 授权发布层级 | AUTH_PUBLISH_LEVEL | 平铺列表，多选 | 调用接口获取 |
| 适用区域 | APPLICABLE_REGION | 三层树形，多选 | 机关-地区部-代表处 |
| 授权发布组织 | AUTH_PUBLISH_ORG | 三层树形，多选 | 机关-地区部-代表处 |
| 产业 | INDUSTRY | 树形，多选 | 调用接口获取 |
| 业务场景 | BUSINESS_SCENE | 平铺列表，单选 | 调用接口获取 |
| 决策层级 | DECISION_LEVEL | 平铺列表，单选 | 调用接口获取 |
| 文档类型 | DOC_TYPE | 平铺列表，单选 | 建议值：PDF、Word、Excel、图片、其他 |
| 状态 | STATUS | 平铺列表，单选 | 草稿、已发布、已失效 |
| 运算符 | OPERATOR | 平铺列表，单选 | >, <, =, >=, <=, != |
| 计量单位 | COMPARE_UNIT | 平铺列表，单选 | 千万、兆、CNY、USD等 |
| 规则参数字段 | - | 平铺列表，单选 | 调用规则参数接口获取 |

## 5. 状态流转规则

```
┌─────────┐    发布    ┌───────────┐    失效    ┌─────────┐
│  草稿   │ ────────> │  已发布   │ ────────> │  已失效  │
│ DRAFT  │           │ PUBLISHED │           │ INVALID │
└─────────┘           └───────────┘           └─────────┘
     │                      │                      │
     │ 删除                 │ 删除                 │ 删除
     ▼                      ▼                      ▼
  (已删除)               (已删除)               (已删除)
```

### 状态权限控制

| 状态 | 可编辑 | 可保存 | 可发布 | 可失效 | 可删除 |
|------|--------|--------|--------|--------|--------|
| 草稿 | ✓ | ✓ | ✓ | ✗ | ✓ |
| 已发布 | ✗ | ✗ | ✗ | ✓ | ✓ |
| 已失效 | ✗ | ✗ | ✗ | ✗ | ✓ |

## 6. 业务规则

### 6.1 授权书规则
- BR-001: 授权书名称不能重复
- BR-002: 草稿状态可编辑、可删除
- BR-003: 已发布状态不可编辑，可失效、可删除
- BR-004: 已失效状态不可编辑，可删除
- BR-005: 发布前校验基本信息完整性

### 6.2 场景规则
- BR-006: 场景名称在同一个授权书下不能重复
- BR-007: 规则配置和问卷至少配置一项
- BR-008: 条件组支持嵌套，嵌套层级无限制

### 6.3 规则参数规则
- BR-009: 规则参数名称和英文名不能重复
- BR-010: 数据类型为"比对字段"时，需要指定关联字段

### 6.4 附件规则
- BR-011: 附件大小限制（由前端控制）
- BR-012: 加密功能暂不实现

### 6.5 场景匹配规则
- BR-013: 命中几个场景返回几个
- BR-014: 返回命中场景的决策层级
- BR-015: 条件默认使用"且"逻辑连接
